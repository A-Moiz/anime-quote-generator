# 🎌 Anime Quote Generator

A simple and interactive web application that displays random anime quotes and allows users to browse and search through all available quotes.

The project uses Firebase Firestore as a database and is deployed on Netlify.

---

## 🌐 Live Website

🔗 **View the published site here:**  
[Anime Quote Generator](https://anime-quote-generate.netlify.app)

---

## ✨ Features

- 🎲 Displays a random quote on page load
- 🔄 "Next Quote" button to generate new random quotes
- 🖼 Displays character image with each quote
- 📚 View all quotes in a list format
- 🔍 Live search functionality
  - Filter by **character name**
  - Filter by **anime name**
- 🌙 Dark / Light mode toggle
- ☁️ Quotes stored in Firebase Firestore
- 🚀 Deployed using Netlify

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Firebase Firestore
- Netlify (Deployment)

---

## 🔥 How It Works

- On page load, a random quote is fetched from Firestore.
- Users can generate new quotes using the button.
- The search bar dynamically filters quotes by character or anime.
- All quotes are rendered dynamically using JavaScript DOM manipulation.
